This is a custom factorial function which is able to handle decimal numbers. It uses the gamma function for positive real parts of a complex number, 
or whats called: Euler integral of the second kind.
I also included a version of it so that you can use in your own projects.
To use this:
1. Have the file in the same folder as your project
2. Import fact
3. Use fact.fact(x) where x is the number you want the factorial of
